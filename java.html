Connect 4

Information

A traditional 2 player Connect 4 game. Clicking 'Start Game' randomizes which player goes first and makes the buttons at the top of each column enabled. The current player clicks the button at the top of the column where they want their piece to go. Once a column is full, the button becomes disabled. The game continues until one player has 4 in a row or there are no more spots available, which results in a tie.

Built With

  • Java
  • Swing
  • Intellij IDEA
  • GUI Form

Tic-Tac-Toe

Information

A traditional 2 player Tic-Tac-Toe game. Clicking on a square will turn the square to either an X or O depending on whose turn it is. Once the square has been clicked, it becomes disabled so as not to allow for the square to be changed. The display at the top shows the current player's turn. At the bottom, you can see the number of wins for X and O as well as ties. Once a player scores a Tic-Tac-Toe, or a tie occurs, all buttons are disabled, the winning squares (if any) are highlighted, the score is updated, and the clear button changes to "Start Game".

Built With

  • Java
  • Swing
  • Intellij IDEA
  • GUI Form

Additional Features Added

  • Combo box with color options

Banking Application

Information

Banking application that is connected to a PostgreSQL database. Users login to their account with their account number which is stored in the database. Once logged in, users can check their balance, make a deposit, or withdraw money from their account. If the user attempts to withdraw more money than is in their account, the transaction will not go through, and they will get an error message on the screen. Each successful transaction is updated in the database.

Built With

  • Java
  • Swing
  • Intellij IDEA
  • GUI Form
  • PostgreSQL
